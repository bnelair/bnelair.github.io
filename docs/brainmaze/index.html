BrainMaze Project#

Functionalities#

  • EEG Signal Processing: Includes transformations, classification, and feature extraction.

Submodules#

BrainMaze Utils#

  • Repository: Repo

  • Documentation: Documentation

  • Description: Elementary utilities used across multiple BrainMaze projects and codes. This was separated to ensure lightweight deployment and package controllability.

BrainMaze ZMQ#

  • Repository: Repo

  • Documentation: Documentation

  • Description: Wrappers around ZMQ controlling for optimal socket release after process termination.

Acknowledgment#

This code was developed and originally published by Mivalt (2022) and Sladky (2022). When using this toolbox, please cite the related papers. See the sections below for references to individual submodules.

  • F. Mivalt et V. Kremen et al., “Electrical brain stimulation and continuous behavioral state tracking in ambulatory humans,” J. Neural Eng., vol. 19, no. 1, p. 016019, Feb. 2022, doi: 10.1088/1741-2552/ac4bfd.

  • V. Sladky et al., “Distributed brain co-processor for tracking spikes, seizures and behaviour during electrical brain stimulation,” Brain Commun., vol. 4, no. 3, May 2022, doi: 10.1093/braincomms/fcac115.

  • F. Mivalt et V. Sladky et al., “Automated sleep classification with chronic neural implants in freely behaving canines,” J. Neural Eng., vol. 20, no. 4, p. 046025, Aug. 2023, doi: 10.1088/1741-2552/aced21.
