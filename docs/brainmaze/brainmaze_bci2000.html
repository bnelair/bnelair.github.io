brainmaze_bci2000#

# BrainMaze - Codes for CorTec and BCI2000

Tools for converting BCI2000 data to [BIDS](https://bids.neuroimaging.io/). Currently supports conversion between CorTec and BIDS using [Mef3 format ](bnelair/mef_tools).

## Installation

`bash pip install --force git+https://github.com/bnelair/brainmaze_bci2000.git@dev `
