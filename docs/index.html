Bioelectronics Neuropysiology and Engineering Lab (BNEL)#

This is the central documentation for software tools developed in our Bioelectronics Neuropysiology and Engineering Lab (BNEL) at Mayo Clinic. The tools are designed to facilitate the analysis of brain electrophysiology, behavior, and dynamics.

BrainMaze: Brain Electrophysiology, Behavior and Dynamics Analysis Toolbox

A Python toolbox with tools for processing of intracranial iEEG recordings.
